## AgenticHelper Minecraft Plugin

AgenticHelper добавляет на сервер Paper/Spigot:

- персонализированное приветствие при заходе игрока;
- ежедневный подарок командой `/agentgift`;
- безопасное хранение информации о полученных подарках через `PersistentDataContainer`.

### Установка

1. Скомпилируйте плагин (см. ниже) или скачайте готовый JAR из веб-приложения.
2. Поместите `AgenticHelper.jar` в папку `plugins` вашего Paper/Spigot сервера.
3. Перезагрузите или перезапустите сервер.

### Сборка вручную

```bash
./build.sh
```

Готовый файл появится в `build/AgenticHelper.jar`.

### Конфигурация

Файл `config.yml` создаётся автоматически при первом запуске. Вы можете изменить приветствие, используя местоимещение `%player%` для имени игрока.
